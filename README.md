# DESAFIO DOCKER FULLCYCLE

## Build project

<code>docker-compose buid</code>

## Up containers

<code>docker-compose up -d</code>

## Access the container

### nginx
<code>
docker exec -it nginx bash
</code>

### node
<code>
docker exec -it app bash
</code>

### mysql
<code>
docker exec -it db bash
</code>
